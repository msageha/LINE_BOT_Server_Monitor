### 通知機能
- 最終ログインから1週間立つと，通知
- CPU, memory, HDD, トラフィック使用率が，(極端に)変化したら通知
- 変なアクセスがあったら通知(sshなど)
- ファンの停止などがあったら通知
- その他，各種SNMP関連

- プロセス確認
- ログインユーザー確認
